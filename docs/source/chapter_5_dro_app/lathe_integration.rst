===================================
Digital indexer - LATHE integration
===================================

The Digital Readout (DRO) is a high-performance application for manual lathes and milling machines.

* Real-time axis tracking with low latency.
* Wireless data transmission to Android or Windows tablets.
* Robust mounting for machine shop environments.

Implementation Steps
--------------------
1. Flash the specialized DRO firmware via ST-Link.
2. Pair the Bluetooth module with the display device.
3. Configure scaling factors for your linear scales.
